🚀 使用方法
-------------------
1. 环境检查:
   cd cycronet-build
   .\check_env.ps1

2. 编译所有平台:
   .\build_all.ps1 -Platform all

3. 或单独编译:
   .\build_all.ps1 -Platform windows
   .\build_all.ps1 -Platform linux
   .\build_all.ps1 -Platform macos

4. 测试安装:
   pip install target\wheels\cycronet-*.whl

📝 重要说明
-------------------
1. Linux 编译需要 Docker Desktop 运行
2. macOS 交叉编译使用 Zig 工具链
3. 不要修改 cronet-libs/ 和 cronet-bin/ 目录
4. 所有路径使用相对路径,可移植到任何位置
